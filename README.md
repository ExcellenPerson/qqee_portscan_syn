# qqee_portscan_syn

> 跨平台端口扫描引擎。剥离于自己的代理服务器扫描工具。

> 请勿用于违法攻击 !

> [http://www.qq.ee](http://www.qq.ee/)

***

> 特点

·1  使用syn握手包，节省流量且不会被目标记录。

·2  自动对抗防火墙，自动响应常见防火墙验证数据包。

·3  每个端口状态仅用 1bit，同时扫描100万个ip占4.7MB内存，扫1亿个ip占内存470MB。

·4  超时自动重试，被墙自动跳过该ip。

·5  多线程 + 原子操作。

***

windows平台编译需添加库 -lws2_32 -lkernel32 -lmswsock
